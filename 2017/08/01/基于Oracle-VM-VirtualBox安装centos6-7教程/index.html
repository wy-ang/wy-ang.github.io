基于Oracle VM VirtualBox安装centos6.7教程

发表于 2017-08-01

人的大脑适合创造，不适合记忆，所以撰写此文档以便日后查阅

作为Linux小白，安装Linux环境是学习Linux的第一步,Oracle VM VirtualBox虚拟机的安装这里就不讲了，网上有教程，而且安装简单。

第一步 ：新建虚拟电脑

点击Oracle VM VirtualBox虚拟机（以下简称OVM）左上角的新建按钮弹出新建虚拟电脑窗口（如下图）

名称、类型、版本的选项就按照上图中的设置即可，设置完之后点击下一步跳转到如下图窗口

这一步根据自己的需求自行设置内存大小,一般默认就好（软件已经根据电脑内存自动分配好了），点击下一步跳转到如下图窗口

选择现在创建虚拟硬盘后点击创建，跳转到创建虚拟硬盘窗口（如下图）

默认选择第一个然后点击下一步跳转到如下图窗口

选择固定大小，点击下一步跳转到如下图窗口

上图中第一个文本框内的文件名可以随便输（建议英文），第二个设置虚拟硬盘大小可根据自己的实际情况设置，也可以默认（软件默认已经根据电脑的硬盘情况分配好了），设置好后点击创建，然后会弹出一个创建虚拟硬盘的进度条（如下图），等待两分钟左右。

第一步 ：新建虚拟电脑结束

第二步：设置虚拟电脑系统

我们创建好虚拟电脑后，就要开始设置虚拟电脑啦。

右键点击已经创建好的虚拟电脑，选择设置（如下图）

点击设置弹出虚拟电脑设置窗口,然后点击系统（如下图）

上图中各设置项按照图中的设置即可（注意启动顺序的位置一定要按照图中的位置设置不能乱），点击OK保存，然后选择网络（如下图）

连接方式选择桥接网卡，点击OK保存，然后选择存储（如下图）

点击控制器，然后点击第一个加号，然后弹出选择磁盘窗口（如下图）

点击选择磁盘，选择电脑中的centos文件（如下图）

选择完碟片后点击OK。

第二步：设置虚拟电脑系统结束

第三步：安装系统

选中创建的虚拟电脑，然后点击左上角的启动会弹出如下图窗口

选择第一个，然后跳转如下图（等待两三分钟，喝杯茶）

上图走完后会跳转到如下图所示界面

选择skip（跳过），然后出现如下图所示界面（等待两三分钟）

上图走完之后，界面就跳转到如下图所示

点击next

选择简体中文，点击next

选择美国英语式，点击下一步

选择基本存储设备，弹出存储设备警告窗口（如下图）

选择是，忽略所有数据

起个名字，继续下一步，

默认，下一步

设置你的根账号和密码（请牢记你的账号和密码，很重要）

选择替换现有的Linux系统，然后下一步

选择将修改写入磁盘，然后就让它自己跑吧（再喝一杯茶）

恭喜你大功告成

可以开始书写优雅的命令啦^_^
