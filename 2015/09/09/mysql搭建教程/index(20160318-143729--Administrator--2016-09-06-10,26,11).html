  • 官网下载

  • 免安装版的，所以解压地址随便；
    1.找到你解压的路径（如：D:\mysql-5.6.26-winx64）；
    2.新建my.ini文件;配置如下（D:\mysql-5.6.26-winx64改为你自己的路径）

  • 管理员运行cmd

  1. 启动数据库：net start mysql
  2. 停止数据库：net stop mysql
  3. 卸载：sc delete mysql

  4. 进入数据库：mysql -uroot -p

    注：第一次没有密码，直接回车

  5. 设置数据库密码：
    mysql>set password =password(‘你的密码’);
    mysql>flush privileges;

  6. 退出数据库：exit
